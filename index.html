Comandos Gits

Nesta página temos uma sequencia de comandos Gits!

Comando Git Init: Inicializa o git dentro de uma pasta

Comando Git status:Mostra o status como estar os arquivos dentro da pasta do git

Comando Git config --global user.nome "nome"Configura o nome de usuario no git

Comando Git config --global user.email "email"

Comando Git add mais o nome do aquivoEnvia apenas o arquivo que foi passado o nome

Comando Git add .: Adiciona todos os arquivos

Comando Git commit -m "messagem do commite":: faz o commit e grava a messagem a respeito do commit

Comando Git log : Mostra o log do commit

Comando Git branch nome: Cria uma nova branch

Comando Git checkout nome da branch: Faz a troca da branch
